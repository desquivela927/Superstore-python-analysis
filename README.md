# Global Superstore — Python Analysis

## Description
Exploratory data analysis of the Global Superstore dataset with 51,290 orders using Python, Pandas and Matplotlib/Seaborn.

## Tools
- Python (Google Colab)
- Pandas - data analysis
- Matplotlib + Seaborn - data visualization
- Dataset: Global Superstore (Kaggle)

## Analysis performed
1. Total sales by category
2. Profit by region
3. Sales trend by year
4. Top 10 products by sales
5. Executive dashboard with subplots

## Key findings
- Technology leads in sales ($4.7M)
- Central is the most profitable region
- Sales grow consistently year over year
- Apple Smart Phone is the best-selling product
